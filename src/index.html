About

'metadoc' is Editable/Light/Interactive/PlainText documentation system that built with vanilla HTML5 / CSS3 / JavaScript.

*** This document is created with metadoc.

*** For detail, see GitHub.


Usage

Saving

  • Ctrl+Shift+S: Save document to file
  • Ctrl+S: Overwrite document

Document Property

  • Toggle Edit / View mode
  • Edit document title

Editing

  • Change tag type
  • Change tag type to designated type
  • Heading
  • Create table
  • Create link
  • Bullet List , Numbered Bullet List
  • Edit tag
  • Move tag
  • Delete tag
  • Pop (Unwrap) tag

Styling

  • Align text
  • Subscript/Superscript
  • Font size
  • Bold
  • Strike-through
  • Italic
  • Underline
  • Text color
  • Edit style

Globals

  • Edit CSS
  • Edit Script
