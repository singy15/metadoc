#About

Morphdoc is Editable/Standalone/Text-based/Interactive/Light document system.
It is build on pure HTML5/JavaScript with support of Native FileSystem API.

#Usage

- Ctrl+S to overwrite.
- Ctrl+Shift+S to write to another file.
